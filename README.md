# pyqt-custom-titlebar-setter
PyQt Custom Titlebar Setter

## Requirements
* PyQt5 >= 5.15

## Setup
```pip3 install git+https://github.com/yjg30737/pyqt-custom-titlebar-setter.git --upgrade```
